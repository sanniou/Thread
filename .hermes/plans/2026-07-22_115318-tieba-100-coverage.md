## 1. 当前完成度（估）

- **已可用主路径 ~55%**：登录、吧列表、帖列表、详情、楼中楼、回复+图、点赞、趋势、用户帖/回。
- **API 声明 ~90%**，接线与 UI **远低于** API。
- **最大断点**：搜索未注册、`createThread` 关闭、收藏/消息/签到未进产品路径。

## 2. P0 实现清单（本刀）

1. `TiebaSearchConnector : ForumSearchConnector`
   - `searchTopics` → hybrid `searchThread`
   - `searchComments` → 同接口映射 `pid` 为 Comment（Thread UI 的 REPLY 类型）
2. DI：`RuntimeSourceRegistration.search = TiebaSearchConnector`
3. `SourceCapabilities.supportsSearch = true`

5. 文档：CHANGELOG / README / 矩阵勾选 P0

## 4. 风险

- Hybrid 搜索依赖 Referer + Cookie；登录失效时需明确错误。
- `SearchThreadBean` 字段可能缺省；序列化需宽松默认。
- 发主题可能触发验证码；P0 允许失败信息透出。
