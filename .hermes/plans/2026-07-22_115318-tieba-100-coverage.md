# Tieba 100% Coverage Plan

Date: 2026-07-22  
Baseline tip before plan: `21d83e2a` (0.38.0)  
Current: **0.40.0** ships P0 + P1

## Decisions

1. Coverage = business-action parity vs TiebaLite-4.0-dev, not pixel 1:1.
2. Extend via Source / CapabilityConnectors / shared services; UI only consumes capabilities.
3. Existing main paths (login, list/detail, reply, sub-comments, upvote, image upload) must keep absorbing design/UI/API quality.
4. Platform admin (delete thread/post, bawu tools) is **deferred**. Webview external links may remain as optional extension.
5. Ship by version knives: P0 search+createThread, P1 membership/store/inbox, P2 sign/search extras, P3 non-admin parity.
